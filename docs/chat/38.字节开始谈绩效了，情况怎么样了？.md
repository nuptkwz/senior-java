# 字节开始谈绩效了，情况怎么样了？

> 绩效周的氛围犹如高考放榜，有人因一夜“期权自由”狂喜到抱着显示器哭泣，也有人看着“M-”的评级自嘲准备“提桶跑路”。

“字节绩效开奖了，有人财务自由，有人‘负’务自由！”近日，在知名职场社交平台脉脉上，类似的帖子铺天盖地。绩效评定结果一出，字节跳动再次被推上风口浪尖。

这个曾经以“字节一年，人间三年”的超高速增长和极致压力闻名的互联网巨头，如今正经历一场深刻的管理转向。其核心，正从“卷死你”逐渐变为“宠坏你”。

---

# 01 变革

字节跳动正式向全体员工发送了一封题为《加大绩效区分、激励力度，全年绩效评估1.30启动》的邮件。这封邮件如同一颗投入平静湖面的石子，激起了层层涟漪。

其核心变革直指员工最关心的两个问题：**钱怎么发**，以及**人怎么评**。

在激励层面，字节祭出了“真金白银”的组合拳。最引人注目的，是期权归属节奏的全面加速。自2024年起，员工薪酬包中的期权，除入职第一年外，**归属周期从“按年”缩短为“按季度”**。

这意味着员工更快地能将纸面财富转化为实际收益，流动性大大增强。同时，第一年的归属比例也有所提升，进一步增强了早期激励。

年终激励的构成也发生了根本性改变。从2025年1月起，年度激励将明确拆分为“全年奖”和“绩效期权”两部分。**全年奖上限为3个月月薪，而真正拉开差距的，是面向高绩效员工的“绩效期权”**。

这部分期权的计算基数从月薪变为“月薪总包”，并可能在当期价格上额外增发，高绩效员工将获得远超以往的长期回报。公司每年两次的期权回购机制，则确保了这部分资产的变现渠道。

配合激励政策的调整，薪酬结构也同步优化。公司将年终奖月数大于3的方案统一调整为3个月，保持总包不变的同时，**提升员工的月度现金收入**，缓解即时生活压力。

# 02 阵痛

政策的风向标已明确指向“加大区分、重奖优秀”，但具体落到每个个体身上，却是几家欢喜几家愁。

脉脉等社交平台成了这场“绩效开奖”的情绪宣泄场。一位认证为字节员工的用户以戏谑又真实的笔触描绘了办公室的生态：“绩效周简直比高考出分还刺激，办公室里的空气都凝固了，连喝水都要小心翼翼看Leader脸色！”

他将不同绩效档位员工的境遇刻画得入木三分：**拿到最高档“E”的员工**，被形容为“抱着显示器哭得像个两百斤的孩子，不是因为被裁，而是因为奖金多到数不清”，享受着“凡尔赛”式的暴击。

而自评为“M-”（勉强符合预期）的他，则用“心态稳如老狗”来自我解嘲，甚至做好了“提桶跑路”去送外卖的准备，并调侃自己的奖金“是不是被隔壁工位那个天天表演‘无效加班’的同事卷没了”。

这种强烈的对比，正是新政策“加大区分度”意图的直接体现。公司明确表示，**将把更多的激励资源向高绩效员工倾斜**，让优秀的回报变得“肉眼可见”。

# 03 安抚

硬币的另一面是“安抚多数”。与以往“绩效不合格就面临淘汰”的紧张氛围不同，今年的绩效沟通似乎多了一丝“人情味”。

一位拥有十年以上经验的HR在脉脉上分析指出，字节正在经历从“卷死你”到“宠坏你”的转变。公司内部传递的信息是，占员工多数的 **“M”档（符合预期）是公司的中流砥柱**，而不再是“平庸”的代名词。

这种定位的转变，极大地缓解了中间层员工的焦虑。更关键的是，对于绩效垫底的“M-”员工，政策也给出了更多转圜空间。

脉脉信息显示，“M-”不一定直接等同于“毕业”（裁员），员工有机会通过“活水计划”（内部转岗）获得新的机会。**晋升门槛也悄然降低**，从过去“需要连续两次M+”调整为“一次M+即可参选”。

这些变化被戏称为“给卷得脱水的打工人，直接拉了一把躺椅过来”。虽然许多员工嘴上仍持观望态度，但行动上已开始调整，周报里“降本增效”、“稳定发挥”成了高频词。

# 04 转向

这一系列动作，绝非简单的“福利大放送”，其背后是字节跳动乃至整个互联网行业在新时代下的战略清醒。

首先，是**对“人才战”成本的重新计算**。过去几年，互联网行业高频的人员流动和激烈的“抢人大战”推高了人力成本，但很多时候，高薪挖来的人未必能长期创造匹配的价值，文化的磨合、团队的稳定性都面临挑战。

字节的新政策，本质上是将激励资源更多地向内部优秀员工沉淀，用更丰厚的长期回报绑定核心人才，降低被竞争对手“挖角”的风险。**把“打工人”变成“合伙人”**，无疑是更高明的留人策略。

其次，是**对管理哲学的迭代**。那个靠极致压力和快速迭代就能赢得一切的时代正在过去。当业务增速放缓，进入深耕和守成阶段，保持团队的稳定性和积极性变得至关重要。

“在这个AI都能写代码的年代，人心才是最难被算法替代的‘核心资产’。”脉脉上这位HR的观点一针见血。与其用高压让人才身心俱疲、被动流失，不如建立更健康、更可持续的激励生态，**实现公司与员工的“和和气气把钱赚了”**。

最后，这也是对**宏观经济与行业周期变化的务实应对**。在投资市场趋于谨慎、业务扩张更注重ROI的背景下，互联网大厂普遍从规模优先转向效率优先。

字节的绩效改革，正是通过更精细化的激励杠杆，驱动组织将每一分力气都用在刀刃上，实现“降本增效”的深层目标。

---

字节跳动此次绩效变革，标志着这家顶级互联网公司的管理文化进入 **“后疯狂增长时代”** 的新阶段。它不再仅仅用单纯的薪酬数字来竞争，而是开始构建一个更具粘性、更关注长期价值与员工心理预期的综合激励生态系统。

对员工而言，“E档”的狂欢是肉眼可见的财富机遇，而“M档”的安抚则提供了难得的安全感与喘息空间。这是一场从“恐惧驱动”到“希望驱动”的转变。

当大厂开始认真“宠”员工，其背后是对“人心”这项终极核心资产的重新定价。这场发生在字节工位上的绩效变革，或许正是中国互联网职场文化从青春期走向成熟期的一个缩影。
