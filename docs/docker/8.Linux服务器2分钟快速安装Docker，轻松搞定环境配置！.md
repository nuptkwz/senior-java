## 1. 快速安装Docker

安装Docker其实非常简单，只需一条命令即可搞定。打开你的Linux服务器终端，执行以下命令：

```
sudo yum install docker-ce docker-ce-cli containerd.io
```

如果你使用的是Ubuntu系统，可以使用以下命令：

```
sudo apt-get install docker-ce docker-ce-cli containerd.io
```

**小贴士**：在安装前，建议先更新系统软件包以确保兼容性。可使用`sudo yum update`（CentOS）或`sudo apt-get update`（Ubuntu）进行更新。

## 2. 配置Docker镜像加速

安装完成后，我们需要进行一些配置优化。首先，在/etc/docker目录下创建或修改daemon.json文件：

```
{
"storage-driver":"overlay2",
"storage-opts":["overlay2.override_kernel_check=true"],
"insecure-registries":["0.0.0.0/0"]
}
```

**配置说明**：
- **overlay2**是Docker推荐的存储驱动程序，性能更优
- **insecure-registries**配置允许Docker访问非HTTPS的镜像仓库，在内网环境中特别实用

对于国内用户，强烈建议配置**镜像加速器**以提升镜像下载速度。在daemon.json文件中添加以下内容（以阿里云镜像加速器为例）：

```
{
"registry-mirrors": ["https://jb34uuwf.mirror.aliyuncs.com"],
"storage-driver":"overlay2",
"storage-opts":["overlay2.override_kernel_check=true"],
"insecure-registries":["0.0.0.0/0"]
}
```

你可以根据自己使用的云服务商选择相应的镜像加速地址。

## 3. 验证安装与启动

配置完成后，需要重新加载配置并重启Docker服务：

```
systemctl daemon-reload
systemctl restart docker
```

接下来，让我们验证Docker是否安装成功：

```
docker version
```

此命令将显示Docker的版本信息，如果正常显示，说明安装成功。

为了进一步验证Docker能否正常运行容器，可以尝试运行一个测试容器：

```
docker run hello-world
```

如果看到欢迎信息，恭喜你！Docker已经成功安装并运行。

## 4. 设置Docker开机自启

为了方便使用，建议将Docker设置为开机自启：

```
systemctl enable docker
```

## 5. 常见问题与技巧

**权限问题**：如果每次执行docker命令都需要sudo，可以将当前用户加入docker组：

```
sudo usermod -aG docker $USER
```

执行后需要**重新登录**或执行`newgrp docker`命令生效。

**镜像下载缓慢**：如果感觉镜像下载速度慢，请检查前面提到的镜像加速器配置是否正确。

**存储驱动选择**：overlay2是当前推荐的存储驱动，但如果你遇到存储相关问题，可以考處使用devicemapper等其他驱动。
